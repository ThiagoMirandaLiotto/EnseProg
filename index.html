- ¿Que es HTML?

HTML (HyperText Markup Language) es el lenguaje de marcado estándar
utilizado para crear y estructurar el contenido de
las páginas web. Es la base fundamental de cualquier sitio web y
se utiliza para describir la estructura y presentación de la
información que se muestra en un navegador web.
Basicamente define la ESTRUCTURA de la página web

- ¿Que es CSS?


CSS (Cascading Style Sheets) es un lenguaje de diseño utilizado
para controlar la presentación y el formato de un documento HTML.
Se utiliza para definir cómo se deben mostrar los
elementos en una página web, incluyendo aspectos
como el diseño, colores, fuentes y márgenes.
Basicamente define el DISEÑO de una página web
